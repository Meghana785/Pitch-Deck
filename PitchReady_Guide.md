# PitchReady: Mult-agent AI Web Application
## Official Setup and Execution Guide

Welcome to the **PitchReady** setup guide. This document explains the steps to configure and run the PitchReady mult-agent AI web application, which consists of a Next.js Frontend, a FastAPI Backend, and a LangGraph Worker.

---

### Prerequisites

Before starting, ensure you have the following installed on your machine:
- **Node.js** (v18 or higher) and **npm**
- **Python** (v3.10 or higher) and **pip**
- **Docker** and **Docker Compose**
- External services credentials:
  - **Neon Auth** credentials
  - **PostgreSQL** database (Asyncpg)
  - **Upstash Redis** (for the Langgraph worker and backend queues)
  - **AWS S3** Credentials (Boto3)
  - **Anthropic API Key** (for Langgraph worker)

### 1. Environment Configuration

Proper environment variables must be configured for the backend, worker, and frontend.

#### Backend and Worker (`.env` file)
Create a `.env` file in the root directory (`/Users/apple/WorkSpace/orbrot/Projects/PitchReady/.env`) with your configuration:

```env
# Database
DATABASE_URL=postgresql://user:password@hostname/dbname

# Upstash Redis
REDIS_URL=rediss://default:password@hostname:port

# AWS S3 (boto3)
AWS_ACCESS_KEY_ID=your_access_key
AWS_SECRET_ACCESS_KEY=your_secret_key
AWS_REGION=us-east-1
S3_BUCKET_NAME=your_bucket_name

# Auth (python-jose)
SECRET_KEY=your_jwt_secret
ALGORITHM=HS256

# Anthropic API Key (worker)
ANTHROPIC_API_KEY=your_anthropic_api_key
```

#### Frontend (`frontend/.env` file)
Navigate to the `frontend/` directory and configure the environment variables required for Next.js, Neon Auth, and API URLs.

```env
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_NEON_AUTH_URL=...
```

### 2. Running the Backend and Worker

The backend services (FastAPI API, LangGraph Worker, and Nginx reverse proxy) are containerized and can be launched together using Docker Compose.

1. Open your terminal and navigate to the root directory of the project:
   ```bash
   cd /Users/apple/WorkSpace/orbrot/Projects/PitchReady
   ```
2. Build and start the Docker containers in detached mode:
   ```bash
   docker-compose up --build -d
   ```
   *This command will start:*
   - **nginx** on ports `80` and `443`
   - **backend** (FastAPI) on port `8000`
   - **worker** (LangGraph queue processor)

3. Verify the containers are running correctly:
   ```bash
   docker-compose ps
   docker-compose logs -f
   ```

*(Alternative Local Backend Run)*:
If you prefer to run the FastAPI backend locally without Docker:
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 3. Running the Frontend

The PitchReady frontend is built with Next.js and Tailwind CSS.

1. Open a new terminal window and navigate to the frontend directory:
   ```bash
   cd /Users/apple/WorkSpace/orbrot/Projects/PitchReady/frontend
   ```
2. Install the Node.js dependencies:
   ```bash
   npm install
   ```
3. Start the Next.js development server:
   ```bash
   npm run dev
   ```
4. Access the frontend application by opening your browser to:
   [http://localhost:3000](http://localhost:3000)
